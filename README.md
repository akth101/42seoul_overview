# 42seoul_overview

42seoul이란?
- 교수, 교재, 학비가 없는 3無 소프트웨어 교육
- 운영 측에서는 과제를 부여할 뿐 모든 것은 학생 스스로 그리고 동료와 함께 해결해야 함
- 정해진 기간 내에 과제를 달성하지 못하면 퇴교 조치되며 과제를 달성하면 퇴교 기한을 늘려줌
- 과제 완성 시 동료 학생 3명의 평가가 이루어지며 한 평가라도 fail일 경우 처음부터 다시 평가를 받아야 함
